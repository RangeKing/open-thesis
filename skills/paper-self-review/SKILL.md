# Paper Self-Review (Chinese Thesis Mode)

中文学位论文提交前质量核验技能。

## 触发场景

- “论文自审”
- “提交前检查”
- “格式是否符合国标”
- “答辩前还有哪些风险”

## 核心检查项

1. 结构完整性
- 封面/声明/中英摘要/目录/正文/参考文献/附录/致谢/成果/作者简介（博士）

2. 摘要与关键词
- 硕士：500-1000 字
- 博士：1000-2000 字
- 关键词 3-8 个，中英对应

3. 排版
- A4、宋体小四、黑体小二、Times New Roman
- 固定行距 20 磅、首行缩进 2 字符

4. 引用
- GB/T 7714-2015 顺序编码制 `[1][2]...`
- 文内文末一致
- 文献数量与时效达标

5. 学术表达
- 避免“本文认为/笔者认为”
- 避免宣传化与模糊归因
- 结论须明确新见解

6. 答辩准备
- 是否具备 Major/Minor 分类回复草稿
- 是否具备“意见-修改-证据位置”映射

## 输出模板

```markdown
## 论文自审报告
- 总体结论：通过 / 有条件通过 / 不通过
- 高风险问题：N
- 中风险问题：N
- 低风险问题：N

### [High] 问题
- 现象：...
- 影响：...
- 修复建议：...
- 截止时间建议：...
```

## 输出约束

- 必须输出可执行修复清单
- 对结构与引用问题，必须给出规范依据（GB/T 条款级别）
