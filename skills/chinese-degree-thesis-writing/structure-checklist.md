# 中文学位论文结构核验清单

## 一、前置部分

- [ ] 中英文封面齐全，信息一致（题目、作者、学院、专业、导师、日期）
- [ ] 原创性声明与使用授权书已放置在摘要前后并签署
- [ ] 中文摘要符合字数要求（硕士 500-1000；博士 1000-2000）
- [ ] 英文摘要与中文摘要内容一致
- [ ] 关键词数量为 3-8，且中英关键词对应

## 二、正文结构

- [ ] 第一章为绪论/引言，明确研究背景、问题、目标、贡献
- [ ] 单列“文献综述”章节，包含“述”与“评”
- [ ] 研究方法章节可复现（数据、流程、参数、评价指标）
- [ ] 结果与分析章节包含对比、消融或稳健性讨论
- [ ] 结论章节给出明确新见解与后续工作

## 三、参考文献（GB/T 7714-2015）

- [ ] 使用顺序编码制 `[1][2]...`
- [ ] 文内引用序号与文末条目一一对应
- [ ] 硕士不少于 40 篇；博士不少于 100 篇
- [ ] 中文与外文比例大致各半
- [ ] 近 5 年文献不少于 1/3
- [ ] 文献类型标识完整（J/M/C/D/R 等）

## 四、附加部分

- [ ] 附录内容完整（公式推导、附表、补充实验）
- [ ] 致谢语言规范、客观
- [ ] 攻读期间成果列表可核验
- [ ] 博士论文包含作者简介

## 五、排版检查

- [ ] A4 纸，正文宋体小四
- [ ] 章标题黑体小二
- [ ] 英文统一 Times New Roman
- [ ] 固定行距 20 磅
- [ ] 首行缩进 2 字符
- [ ] 标题编号符合“第一章”“2.1”等层级规范

## 六、答辩准备

- [ ] 已形成“答辩意见-证据-修改位置”三列表
- [ ] 已准备 Major/Minor 分类回复草稿
- [ ] 所有回复均给出实质改动证据（页码/章节/图表）
